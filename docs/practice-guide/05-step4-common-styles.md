# 4단계: 공통 스타일 작성

`css/style.css`에 Reset, 컨테이너, 헤더, 히어로, 섹션, 푸터 등 공통 스타일을 작성합니다.

---

## 학습 목표

- CSS Reset 적용
- 레이아웃(container, header, main, footer) 스타일
- 히어로·상품 섹션·Swiper 영역 공통 스타일

---

## 실습 방향

1. **Reset**: `*`, `*::before`, `*::after` box-sizing, margin, padding
2. **body**: font-family, color, background (variables 사용)
3. **.container**: max-width, margin auto, padding
4. **.header**: sticky, border-bottom, header-inner grid
5. **.sr-only**: 접근성용 스크린리더 전용 텍스트
6. **히어로·상품 슬라이드**: swiper-container, section 제목
7. **.footer**: 푸터 레이아웃

프로젝트의 `css/style.css`를 참고하여 위 항목을 순서대로 작성합니다. variables.css를 import한 뒤 변수를 사용합니다.
