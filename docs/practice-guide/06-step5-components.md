# 5단계: 컴포넌트 스타일

`css/components.css`에 버튼, 상품 카드, 폼, 배지, 장바구니 모달, Swiper 네비게이션 등 컴포넌트 스타일을 작성합니다.

---

## 학습 목표

- 버튼(.btn, .btn-primary, .btn-outline, .btn-secondary) 스타일
- 상품 카드(.product-card, 배지, 가격, 평점) 스타일
- 폼(.form-group, input, select) 스타일
- 장바구니 모달(.cart-modal-backdrop, .cart-modal) 스타일
- Swiper 네비게이션 버튼 스타일(필요 시)
- `<app-header>`, `<app-footer>`가 사용하는 헤더/푸터 구조 유지

---

## 실습 방향

1. **Buttons**: .btn, .btn-primary, .btn-outline, .btn-secondary, .btn-full, :disabled, :focus-visible
2. **Product card**: .product-card, .product-card-image, .product-card-badges, .badge, .product-card-price, .product-card-rating, .product-card-soldout, .product-card-add-btn
3. **Forms**: .form-group, label, input, textarea, select, focus 스타일
4. **Cart modal**: .cart-modal-backdrop, .cart-modal, .cart-modal-header, .cart-modal-body, .cart-modal-footer
5. **Breadcrumb**(상품 상세용): .breadcrumb, .breadcrumb-link
6. **AppHeader/AppFooter**: 헤더/푸터 마크업을 컴포넌트에서 재사용하므로 기존 클래스 스타일을 그대로 유지

프로젝트의 `css/components.css`를 참고하여 순서대로 추가합니다.

---

- [ ] 저장 후 메인·상품 목록·장바구니 담기 모달 확인
