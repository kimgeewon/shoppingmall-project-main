# 6단계: 반응형 디자인

`css/responsive.css`에 미디어 쿼리를 작성하여 모바일·태블릿·데스크톱에 맞게 레이아웃을 조정합니다.

---

## 학습 목표

- 브레이크포인트 이해 (767px, 768px, 1024px)
- 모바일: 1열, 헤더 검색·메뉴 접기, 상품 그리드 1~2열
- 태블릿·데스크톱: 2~4열 그리드

---

## 실습 방향

1. **@media (max-width: 767px)**: .container padding 축소, 헤더 레이아웃 조정, 모바일 메뉴 표시/숨김, 상품 그리드 1열, 히어로·매거진 세로 배치
2. **@media (min-width: 768px) and (max-width: 1023px)**: 상품 그리드 2열
3. **@media (min-width: 1024px)**: 상품 그리드 3~4열

프로젝트의 `css/responsive.css`를 참고합니다. 개발자 도구 디바이스 툴바로 375px, 768px, 1024px에서 확인합니다.
